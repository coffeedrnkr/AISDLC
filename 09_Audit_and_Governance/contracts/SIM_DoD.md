# Definition of Done: Simulation Agent
**Agent ID:** `SIM_AGENT`
**Artifact:** Simulation Report

## 1. Completeness Checks (The Inventory)
*   [ ] **Persona Match:** Do the simulated users match the PRD Target Audience?
*   [ ] **Scenario Coverage:** Did they walk through the core Happy Paths?

## 2. Quality Checks (The Standard)
*   [ ] **Sentiment Score:** Is the average User Sentiment > 4.0/5.0?
*   [ ] **Frustration Points:** Are friction points identified and logged?

## 3. Traceability Checks (The Red Thread)
*   [ ] **Linkage:** Report links back to the Epic/Story being simulated.
